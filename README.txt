== Installation ==

= Requirements =
* PHP 5.2 or higher
* WordPress v3.0.x or higher

= Installation Process =
* Log into the WordPress admin for your site
* Go to 'Plugins' -> 'Add New'
* Search for 'myWidget recommendations'. Once found click on 'Install Now'
* Select 'Activate Plugin' from the download page or go to 'Plugins' -> 'Installed Plugins' -> 'Activate Plugin'

= Deploy widgets =
* Go to Appearance -> Widgets
* Drag the 'myWidget Recommendations' widget to where you would like it to be located
* Enter the 'cid'
* Click on 'Save'
* Go take a look at your site!
